Process & log

Ingests pasted streams against the ML, auto-logs every fighter ≥1% edge per stream, and saves the card. Re-running for the same card updates instead of duplicating.

When re-pasting ML, refresh existing pending bets in:

UFC Model · weighted blend by source

Per-fighter blend of Saber / MMAEye / Action / Arena. Weights renormalize when a source is missing for a fighter. Min 2 sources to show.

Weight backtest · what would these weights have done?

Replays every graded bet across all cards with the current weight settings — deduped by fighter + card so the same fighter only counts once even if logged in multiple streams. Missing source values get pulled from the card's source maps so the blend uses every available source. "Edge ≥1%" means the blended % beat the market line by ≥1pp. Tune sliders and watch this update live.
